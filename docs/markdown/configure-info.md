## 配置参数

### 基本通用参数
参数名称 | 说明 | 类型 | 默认值 | 是否必须
--------|-----|------|--------|----
type | 组件类型 | string | | 必须 
name | 组件名称，唯一值，组件间交互和调用需使用此名称 | string | | 必须 
config | 组件本身的配置，具体见 [`Components`](#/Component/) | Object | | 必须
