> 以下库文件分为三个版本：
> * dev 版：为当前研发版本，可对遇到的问题进行及时响应并修复，前期推荐使用dev版。

> * stable 版：屏蔽了 warning 及部分 error 问题，使代码尽量可以执行。同时，代码进行了压缩，文件较小，报错及问题定位较困难。
